# Predictive Maintenance Chatbot
AI-powered chatbot for machine failure prediction using Machine Learning and Function Calling.
## Features
- Machine Failure Prediction
- Function Calling Tool
- Gemini Version
- Gemma Version
- Maintenance Recommendations
## Input
- Machine Type
- Air Temperature
- Process Temperature
- RPM
- Torque
- Tool Wear
## Example
Input:
M,298.1,308.6,3000,90,300
output:
Failure Risk: 73%
Recommendation: Immediate maintenance required.
### Gemini Example
Question:
What factors increase machine failure risk?
Answer:
Machine failure risk is commonly affected by tool wear, torque, rotational speed, and operating temperatures.
